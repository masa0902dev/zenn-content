# Markdown & Latex
## snippets
`$$`, `$$\n$$`内は、言語がLatexとして認識される。
よって、数式のSnippet登録はLatexにする必要がある。

settings.jsonで`"[markdown,latex]"`の書き方をすると（おそらくエラーになって）設定が適用されないので注意。

## keybindings
cmd+M : $$
cmd+alt+M : $$\n$$
